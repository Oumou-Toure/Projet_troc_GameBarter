# GameBarter

## Description

GameBarter est une plateforme d'échange de jeux vidéo permettant aux utilisateurs de proposer et de négocier des échanges de jeux entre eux.

Chaque jeu possède un titre, une description, une catégorie, une image et un propriétaire.

Les utilisateurs peuvent proposer un échange en sélectionnant un ou plusieurs jeux et en envoyant un message à un autre utilisateur.
Le destinataire peut accepter, refuser ou répondre afin de négocier.

L'historique des messages est conservé pour chaque échange afin de suivre les négociations.
